Instrução


    Este é o Quiz, um nome de um jogo de questionários para o auto teste. O auto teste é um mecanismo de ajuda que permite ao jogador se auto avaliar. Existem alguns questionários, também denominados de auto teste, que possibilitam identificar se uma pessoa tem comportamento de um jogador patológico. Neste Quiz, você responderá o questionário desenvolvido pelo Jogadores Anônimos do Brasil.


    Sistema de Pontuação


  1. - Resposta afirmativa vale 10 points
  2. - Resposta negativa não pontua
  3. - Você terá 40 segundos para responder cada pergunta
  4. - Caso sua pontuação seja maior ou igual a 70 pontos, você tem tendências a ser um jogador patológico

    5. Comandos


    - Para responder basta clicar na opção Sim ou Não que você será redirecionando para a próxima pergunta
  5. Seta esquerda - Serve para retroceder uma questão

  6. Seta direita - Serve para pular uma questão

  7. Atualizar - Serve para redefinir a pontuação e as perguntas
